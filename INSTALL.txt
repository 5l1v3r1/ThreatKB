
This project uses Yeoman for scaffolding. Specifically, we are using the angular-flask generator. More information on that found here.
https://github.com/rayokota/generator-angular-flask

Pre-requisites
1) Install NPM
2) Install Python 2.7 and virtualenv

Install
0) cd into the main directory.
1) Run install.sh. It will create flask/bin/python
2) flask/bin/pip install -r requirements.txt
3) npm install -g yo
4) npm install -g generator-angular-flask

Run
0) cd into the main directory
1) flask/bin/python run.py
2) grunt server
